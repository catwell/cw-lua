# iatax

![iatax](iatax_logo.png)

## Presentation

iatax is a LÖVE remake of [an old Perl game of mine](http://www.iatax.sf.net).

## How to play?

- Install [LÖVE](http://love2d.org).
- Run the game, for instance on Linux:

    love .
